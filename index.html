OFFICE

Concepto basico de lo que es el office en si, siendo el paquete de programas utilizado para trabajar en una oficina.

MICROSOFT WORD

Concepto basico de word que se utiliza para hacer documentos, cortar, pegar, insertar autoformas, viñetas, interlineado, iluminados, etc.

MICROSOFT EXCEL

Utilizado para hacer mediante el uso de funciones en fórmulas, cálculos avanzados, insertar tablas entre otros.

POWER POINT

Utilizado para hacer presentaciones, insertar autoformas, gif animados, cambiar fuente y tamaño de letra, etc.

ACCESS

Es un programa utilizado para crear bases de datos ya sea para combinar con otros programas o para en si y para guardar datos o informacion.

PUBLISHER

Es un programa para hacer diversos documetos como tarjetas de felicitaciones, trifolios, baner, logos entre otros.

ANGELICA RIVAS
